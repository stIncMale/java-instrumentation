The folder contains external libraries needed for compilation and running the code
as well as licenses for the external libraries.

List of files and folders:
licenses/ - contains licenses for external libraries
asm.jar - asm-1.5.3, used by cglig.jar at runtime
cglib.jar - cglib-2.1_3, required for compilation
javassist.jar - javassist-3.16.1-GA, required for compilation
jcommander.jar - jcommander-1.29, required for compilation
log4j.jar - Apache log4j-1.2.17, required for compilation
