The folder contains sources, binaries, build script, license file and external libraries.

List of files and folders:
>bin/ - contains binaries of examples
>doc/ - contains Javadoc-documentation of examples
>lib/ - contains external libraries needed to compile and run the code
>src/ - contains source files
>build.xml - build script for Apache Ant, tested with ant-1.8.4
>license.txt - WTFPL version 2 license file. All content is licensed under WTFPL version 2
