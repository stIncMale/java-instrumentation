The folder contains sources, build file for Apache Ant, license file and external libraries.

List of files and folders:
>lib/ - contains external libraries needed for compilation and running the code
>src/ - source files
>build.xml - build file for Apache Ant, tested with ant-1.8.4
>license.txt - license file. All content is licensed under WTFPL version 2,
except files or text for which other license is explicitly specified (external libraries for example)
